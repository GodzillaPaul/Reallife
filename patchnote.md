# 退休財務計畫書 V1.27

## Visual Design Upgrade
- 主色系更新為 Indigo：#4f46e5，懸停／強調狀態使用 #4338ca。
- 新增 Amber 輔助色：#f59e0b，用於重要結果與 CTA。
- 全面改為卡片式介面：白底、rounded-2xl、柔和陰影與細邊框。
- 強化 Noto Sans TC 字級與字重層次：主標題、章節標題、數值結果分層更清楚。
- 新增一致化 SVG 圖標，用於姓名、年齡、金額、報酬率與規劃提示。

## UI/UX Upgrade
- 性別選擇改為大型選項卡，取代傳統下拉選單。
- 年齡、退休年齡、平均餘命加入 + / - 步進器。
- 輸入框聚焦狀態改為 Indigo 邊框與光暈。
- 加入「方案一：自己存」資訊提示，說明定存／滾息概念。
- 新增「退休規劃八大好處」區塊，提升首頁說服力與完整度。

## Report / Data Visualization
- 新增退休資金軌跡圖：藍線呈現規劃資金累積，橘色虛線呈現目標退休金。
- 結果頁改為大型數字卡片呈現：目前年齡、退休年齡、退休花費、已準備退休金。
- 退休方案一與方案二改為雙卡片對照，強化「自己存」與「富邦退休規劃方案」差異。
- 保留 V1.26b 方案一利率提示：「假設銀行平均滾息利率為 1.2%」。

## Export / Print
- PDF 與 PNG 匯出按鈕集中於結果頁底部，並增加用途說明。
- 列印樣式優化：隱藏操作按鈕、白底、高對比、保留卡片與圖表清晰度。

## Calculation Patch
- 修正銀行年利率為 0% 時，月複利公式可能產生 NaN 的問題。
